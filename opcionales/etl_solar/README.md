5. ‚úÖ **Ingenier√≠a de caracter√≠sticas:** Creadas 16 caracter√≠sticas derivadas para an√°lisis

### Valores Faltantes Intencionales (Para Pr√°ctica)
**IMPORTANTE:** Este dataset fue dise√±ado PEDAG√ìGICAMENTE para dejar algunos valores faltantes:

- **`module_temp`**: 1 valor faltante (NA) - Simula sensor averiado
- **`temp_diff_module_ambient`**: 1 valor faltante - Derivado de module_temp faltante
- **`temp_excess`**: 1 valor faltante - Derivado de module_temp faltante
- **`dc_ac_ratio`**: 3,007 valores faltantes - Esperado cuando AC = 0 (noches)

**Nota:** El ETL imputa parcialmente algunos valores pero deja otros intencionalmente como NA para que practiques t√©cnicas de imputaci√≥n.

### Errores Intencionales para Detecci√≥n
El dataset contiene **13 tipos de errores** introducidos intencionalmente para pr√°ctica de detecci√≥n:

1. **Temperaturas at√≠picas:** Variaciones de ¬±5¬∞C en temperatura de m√≥dulo
2. **Eficiencias an√≥malas:** Valores multiplicados por 1.5 y eficiencias imposibles (> 2.0)
3. **Ratios DC/AC inconsistentes:** Ratios extremos (5, 15) y f√≠sicamente imposibles (< 1)
4. **Irradiaci√≥n negativa:** Valores f√≠sicamente imposibles de radiaci√≥n solar
5. **Irradiaci√≥n excesiva:** Valores > 2.0 kW/m¬≤ (irrealista)
6. **Inconsistencias t√©rmicas:** Temperatura m√≥dulo < temperatura ambiente
7. **Inconsistencias de potencia:** Potencia AC > potencia DC
8. **Inconsistencias temporales:** Irradiaci√≥n > 0 durante horas nocturnas
9. **Temperaturas extremas:** M√≥dulos > 90¬∞C y ambiente < -15¬∞C
10. **Patrones discretos sospechosos:** Valores de irradiaci√≥n con precisi√≥n artificial

**Ejercicio recomendado:** Usa EDA, boxplots, scatter plots y validaci√≥n de reglas de negocio para detectar estos errores antes del an√°lisis principal.
